## imnNLP :: Transliteration resources directory

This directory contains transliteration pairs, used by imbNLP.Transliteration library

Every transliteration pair set definition should start with the following entries:
auto_capital_case=1; // this tells interpreter to automatically create capital letter pairs
lang_A_id="en1_readme_cyr"; // defines identification for A members of a pair
lang_B_id="en1_readme_lat"; // defines identification for B members of a pair
